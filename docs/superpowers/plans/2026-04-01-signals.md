# Signals Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a "Signals" feature that cross-references DEU summaries with recent news via Google News RSS + Claude analysis to surface positive/negative catalysts per company.

**Architecture:** On-demand GET endpoint fetches Google News RSS, loads company DEU summaries from DB, sends both to Claude which returns structured signals. Two UI surfaces: a `SignalsPanel` on the company detail page, and a dedicated `/signals` dashboard page.

**Tech Stack:** Next.js App Router, Anthropic SDK (claude-sonnet-4-6), Google News RSS (XML), Prisma ORM, shadcn UI, Tailwind CSS, Lucide icons.

### Task 2: Create news fetcher `src/lib/news.ts`

- [ ] **Step 1: Create the Google News RSS fetcher**

Create `src/lib/news.ts`:

```ts
export interface NewsArticle {
  title: string;
  url: string;
  publishedAt: string;
}

export async function fetchNews(companyName: string, ticker: string): Promise<NewsArticle[]> {
  const query = encodeURIComponent(`"${companyName}" OR "${ticker}"`);
  const rssUrl = `https://news.google.com/rss/search?q=${query}&hl=fr&gl=FR&ceid=FR:fr`;

  const response = await fetch(rssUrl, {
    headers: { "User-Agent": "FilingLens/1.0" },
  });

  if (!response.ok) {
    console.error(`News fetch failed: ${response.status}`);
    return [];
  }

  const xml = await response.text();
  const articles: NewsArticle[] = [];

  const itemRegex = /<item>([\s\S]*?)<\/item>/g;
  let match;
  while ((match = itemRegex.exec(xml)) !== null && articles.length < 15) {
    const item = match[1];
    const title = item.match(/<title>([\s\S]*?)<\/title>/)?.[1]?.trim() ?? "";
    const url = item.match(/<link>([\s\S]*?)<\/link>/)?.[1]?.trim() ?? "";
    const pubDate = item.match(/<pubDate>([\s\S]*?)<\/pubDate>/)?.[1]?.trim() ?? "";

    if (title && url) {
      articles.push({
        title: title.replace(/&amp;/g, "&").replace(/&lt;/g, "<").replace(/&gt;/g, ">").replace(/&#39;/g, "'").replace(/&quot;/g, '"'),
        url,
        publishedAt: pubDate,
      });
    }
  }

  return articles;
}
```

- [ ] **Step 1: Create the endpoint**

Create `src/app/api/signals/[companyId]/route.ts`:

```ts
import { NextRequest, NextResponse } from "next/server";
import { db } from "@/lib/db";
import { fetchNews } from "@/lib/news";
import Anthropic from "@anthropic-ai/sdk";
import type { Signal, Theme } from "@/lib/types";

const anthropic = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

export async function GET(
  request: NextRequest,
  { params }: { params: Promise<{ companyId: string }> }
) {
  const { companyId } = await params;

  const company = await db.company.findUnique({
    where: { id: companyId },
    include: {
      summaries: {
        select: {
          theme: true,
          score: true,
          summary: true,
          bulletPoints: true,
        },
      },
    },
  });

  if (!company) {
    return NextResponse.json({ error: "Company not found" }, { status: 404 });
  }

  const articles = await fetchNews(company.name, company.ticker);

  if (articles.length === 0) {
    return NextResponse.json({
      signals: [],
      companyName: company.name,
      analyzedAt: new Date().toISOString(),
      articleCount: 0,
    });
  }

  const themeSummaries = company.summaries
    .filter((s) => s.theme !== "global")
    .map((s) => `**${s.theme}** (score ${s.score}/5) : ${s.summary}`)
    .join("\n\n");

  const articlesList = articles
    .map((a, i) => `${i + 1}. [${a.publishedAt}] ${a.title}\n   URL: ${a.url}`)
    .join("\n");

  const response = await anthropic.messages.create({
    model: "claude-sonnet-4-6",
    max_tokens: 2048,
    messages: [
      {
        role: "user",
        content: `Tu es un analyste financier expert. Tu dois croiser les actualités récentes d'une entreprise avec l'analyse de son Document d'Enregistrement Universel (DEU).

ENTREPRISE : ${company.name} (${company.ticker}) — Secteur : ${company.sector}

RÉSUMÉS DEU PAR THÈME :
${themeSummaries}

ACTUALITÉS RÉCENTES :
${articlesList}

Identifie 3 à 8 signaux en croisant les actualités avec l'analyse DEU. Pour chaque signal :
- Détermine s'il est "positive", "negative" ou "neutral"
- Relie-le à un thème DEU si pertinent (risk, strategy, governance, esg, financial) ou null
- Écris un titre court et percutant
- Écris un résumé de 1-2 phrases expliquant le lien entre l'actualité et le DEU
- Inclus l'URL source de l'article le plus pertinent

Réponds UNIQUEMENT avec un JSON valide, sans markdown :
[{"type":"positive","title":"...","summary":"...","theme":"strategy","sourceUrl":"https://..."}]`,
      },
    ],
  });

  const text =
    response.content[0].type === "text" ? response.content[0].text : "[]";

  let signals: Signal[] = [];
  try {
    const cleaned = text.replace(/```json\n?/g, "").replace(/```\n?/g, "").trim();
    signals = JSON.parse(cleaned);
  } catch {
    console.error("Failed to parse signals JSON:", text);
  }

  return NextResponse.json({
    signals,
    companyName: company.name,
    analyzedAt: new Date().toISOString(),
    articleCount: articles.length,
  });
}
```

### Task 5: Create `SignalsPanel` component

```tsx
"use client";

import { useState } from "react";
import { Button } from "@/components/ui/button";
import { SignalCard } from "./signal-card";
import { Zap } from "lucide-react";
import type { Signal } from "@/lib/types";

interface SignalsPanelProps {
  companyId: string;
  companyName: string;
}

export function SignalsPanel({ companyId, companyName }: SignalsPanelProps) {
  const [signals, setSignals] = useState<Signal[] | null>(null);
  const [loading, setLoading] = useState(false);
  const [articleCount, setArticleCount] = useState(0);

  const analyze = async () => {
    setLoading(true);
    try {
      const res = await fetch(`/api/signals/${companyId}`);
      const data = await res.json();
      setSignals(data.signals);
      setArticleCount(data.articleCount);
    } catch {
      setSignals([]);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div>
      <div className="flex items-center justify-between mb-4">
        <div className="flex items-center gap-2.5">
          <Zap className="w-5 h-5 text-primary" />
          <h2 className="text-2xl">Signaux récents</h2>
        </div>
        {signals === null && (
          <Button onClick={analyze} disabled={loading} size="sm">
            {loading ? "Analyse en cours..." : "Analyser"}
          </Button>
        )}
      </div>

      {loading && (
        <div className="flex items-center gap-2 text-sm text-muted-foreground py-8 justify-center">
          <div className="flex gap-1.5">
            <span className="w-2 h-2 rounded-full bg-primary/50 loading-dot" />
            <span className="w-2 h-2 rounded-full bg-primary/50 loading-dot" style={{ animationDelay: "0.2s" }} />
            <span className="w-2 h-2 rounded-full bg-primary/50 loading-dot" style={{ animationDelay: "0.4s" }} />
          </div>
          Analyse de {companyName} en cours...
        </div>
      )}

      {signals !== null && !loading && (
        <>
          {signals.length === 0 ? (
            <p className="text-sm text-muted-foreground py-4">
              {articleCount === 0
                ? "Aucune actualité récente trouvée."
                : "Aucun signal identifié."}
            </p>
          ) : (
            <div className="space-y-3">
              {signals.map((signal, i) => (
                <div
                  key={i}
                  className="animate-fade-in-up"
                  style={{ animationDelay: `${i * 0.05}s` }}
                >
                  <SignalCard signal={signal} />
                </div>
              ))}
            </div>
          )}
          <p className="text-xs text-muted-foreground mt-4">
            Analyse croisée DEU × actualités ({articleCount} articles). Ne constitue pas un conseil d&apos;investissement.
          </p>
          <Button
            variant="ghost"
            size="sm"
            onClick={analyze}
            className="mt-2 text-xs"
          >
            Relancer l&apos;analyse
          </Button>
        </>
      )}
    </div>
  );
}
```

Create `src/app/signals/page.tsx`:

```tsx
"use client";

import { useEffect, useState } from "react";
import { Button } from "@/components/ui/button";
import { Card, CardHeader, CardTitle, CardDescription, CardContent } from "@/components/ui/card";
import { Badge } from "@/components/ui/badge";
import { SignalCard } from "@/components/company/signal-card";
import { Zap } from "lucide-react";
import type { Signal } from "@/lib/types";

interface CompanyBasic {
  id: string;
  name: string;
  ticker: string;
  sector: string;
}

interface CompanySignals {
  signals: Signal[] | null;
  loading: boolean;
  articleCount: number;
}

export default function SignalsPage() {
  const [companies, setCompanies] = useState<CompanyBasic[]>([]);
  const [signalsMap, setSignalsMap] = useState<Record<string, CompanySignals>>({});

  useEffect(() => {
    fetch("/api/companies")
      .then((r) => r.json())
      .then((data: CompanyBasic[]) => setCompanies(data));
  }, []);

  const analyzeCompany = async (companyId: string) => {
    setSignalsMap((prev) => ({
      ...prev,
      [companyId]: { signals: null, loading: true, articleCount: 0 },
    }));

    try {
      const res = await fetch(`/api/signals/${companyId}`);
      const data = await res.json();
      setSignalsMap((prev) => ({
        ...prev,
        [companyId]: {
          signals: data.signals,
          loading: false,
          articleCount: data.articleCount,
        },
      }));
    } catch {
      setSignalsMap((prev) => ({
        ...prev,
        [companyId]: { signals: [], loading: false, articleCount: 0 },
      }));
    }
  };

  return (
    <div className="max-w-4xl mx-auto">
      <div className="mb-10 animate-fade-in-up">
        <div className="flex items-center gap-2.5 mb-2">
          <Zap className="w-6 h-6 text-primary" />
          <h1 className="text-4xl">Signaux</h1>
        </div>
        <p className="text-muted-foreground text-lg">
          Analyse croisée DEU × actualités récentes.
        </p>
      </div>

      <div className="space-y-4">
        {companies.map((company) => {
          const state = signalsMap[company.id];
          return (
            <Card key={company.id}>
              <CardHeader>
                <div className="flex items-center justify-between">
                  <div>
                    <CardTitle>{company.name}</CardTitle>
                    <CardDescription>
                      {company.sector} — <Badge variant="outline" className="font-mono text-xs">{company.ticker}</Badge>
                    </CardDescription>
                  </div>
                  <Button
                    size="sm"
                    onClick={() => analyzeCompany(company.id)}
                    disabled={state?.loading}
                  >
                    {state?.loading
                      ? "Analyse..."
                      : state?.signals
                        ? "Relancer"
                        : "Analyser"}
                  </Button>
                </div>
              </CardHeader>

              {state?.loading && (
                <CardContent>
                  <div className="flex items-center gap-2 text-sm text-muted-foreground py-4 justify-center">
                    <div className="flex gap-1.5">
                      <span className="w-2 h-2 rounded-full bg-primary/50 loading-dot" />
                      <span className="w-2 h-2 rounded-full bg-primary/50 loading-dot" style={{ animationDelay: "0.2s" }} />
                      <span className="w-2 h-2 rounded-full bg-primary/50 loading-dot" style={{ animationDelay: "0.4s" }} />
                    </div>
                    Analyse en cours...
                  </div>
                </CardContent>
              )}

              {state?.signals && !state.loading && (
                <CardContent>
                  {state.signals.length === 0 ? (
                    <p className="text-sm text-muted-foreground">Aucun signal identifié.</p>
                  ) : (
                    <div className="space-y-3">
                      {state.signals.map((signal, i) => (
                        <SignalCard key={i} signal={signal} />
                      ))}
                    </div>
                  )}
                  <p className="text-xs text-muted-foreground mt-3">
                    {state.articleCount} articles analysés. Ne constitue pas un conseil d&apos;investissement.
                  </p>
                </CardContent>
              )}
            </Card>
          );
        })}
      </div>
    </div>
  );
}
```
